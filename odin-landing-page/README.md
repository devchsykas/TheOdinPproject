# 🎨 odin-landing-page

A modern and responsive landing page built with HTML and CSS. This project focuses on layout design, Flexbox usage, and creating visually appealing web pages.

## 🚀 Features

* Responsive layout using **Flexbox**
* Clean and structured design
* Multiple sections (hero, information, quote, call-to-action, footer)
* Consistent spacing and alignment
* Modern UI styling

## 🛠️ Technologies Used

* HTML5 for structure
* CSS3 (Flexbox) for layout and styling

## 💻 How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Explore the layout and design

## 📚 What I Learned

* How to structure a full web page from scratch
* Using **Flexbox** for alignment and layout control
* Working with sections and reusable design patterns
* Improving visual hierarchy and spacing

## 🌟 Future Improvements

* Make the design fully responsive for all screen sizes
* Add animations and transitions
* Improve accessibility (a11y)
* Convert to a reusable component-based layout
