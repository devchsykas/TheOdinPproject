# 🍽️ Odin Restaurant Page

A dynamic, single-page restaurant website built using HTML, CSS, and vanilla JavaScript. This project focuses on DOM manipulation and using Webpack to bundle assets, allowing the entire page's content to be generated dynamically.

Built as part of [The Odin Project's Full Stack JavaScript Curriculum](https://www.theodinproject.com/).

---

## 🛠️ Tech Stack & Concepts Covered

- **HTML5** (Semantics & dynamic generation)
- **CSS3** (Flexbox/Grid layout and modern styling)
- **Vanilla JavaScript** (ES6 Modules, DOM manipulation, event handling)
- **Webpack** (Asset management, code bundling, development server)
- **NPM** (Package management)

---

## ⚙️ Features

- **Single Page Application (SPA) structure:** The page never refreshes; content updates instantly when switching tabs.
- **Modular Code:** Dedicated JavaScript files handle the rendering for the Home, Menu, and Contact tabs.
- **Tabbed Navigation:** Seamless switching between different views of the restaurant.
- **Responsive Design:** Optimized for both desktop and mobile viewing.

   ```bash
   cd YOUR_REPOSITORY_NAME
   ```

````bash

```bash
````
