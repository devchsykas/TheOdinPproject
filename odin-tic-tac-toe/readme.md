# Tic Tac Toe Game

A simple, browser-based Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Overview

This project is a classic Tic Tac Toe implementation that allows two players to take turns placing X and O on a 3x3 board. The game detects winning combinations, highlights the winning line, and announces a tie when all squares are filled with no winner.

## Features

- Two-player gameplay (player X and player O)
- Win detection for rows, columns, and diagonals
- Tie detection when the board is full
- Reset button to start a new game
- Responsive and clean UI with CSS styling

## Files

- `index.html` — game markup and board structure
- `styles.css` — game styling and layout
- `app.js` — game logic and event handling

## How to Run

1. Open `index.html` in your browser.
2. Click a square to place your marker.
3. Players alternate turns until one wins or the board fills.
4. Click "Reset" to clear the board and play again.

## How to Play

- Player X always goes first.
- Players alternate placing their symbol in an empty square.
- The first player to get three of their symbols in a row (horizontal, vertical, or diagonal) wins.
- If all squares are filled and no player has three in a row, the game ends in a tie.

## Notes

- This project is a great starting point for learning JavaScript game logic, DOM manipulation, and basic front-end development.
- You can extend the game with score tracking, sound effects, or animation.
