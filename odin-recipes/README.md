# 🍽️ Recipes

A simple multi-page recipe website built with HTML. This project focuses on learning the fundamentals of web structure, linking pages, and organizing content.

## 🚀 Features

* Multiple recipe pages connected through links
* Clean and semantic HTML structure
* Use of headings, lists, and images
* Simple and easy navigation between pages

## 🛠️ Technologies Used

* HTML5 for structure

## 💻 How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Navigate through the different recipe pages

## 📚 What I Learned

* How to create and structure basic HTML pages
* Linking multiple pages using relative paths
* Using lists (`ul`, `ol`) for content organization
* Writing semantic and readable HTML

## 🌟 Future Improvements

* Improve layout and spacing
* Make the site responsive
* Add more recipes and images
