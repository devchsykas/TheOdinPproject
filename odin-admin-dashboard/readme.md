# 📊 Admin Dashboard

A modern and responsive admin dashboard built with HTML and CSS. This project focuses on advanced layout techniques using CSS Grid and creating structured, data-driven interfaces.

## 🚀 Features

- Dashboard layout using **CSS Grid**
- Sidebar navigation with multiple sections
- Header with search bar and user info
- Card-based content layout
- Clean and organized UI design

## 🛠️ Technologies Used

- HTML5 for structure
- CSS3 (Grid & Flexbox) for layout and styling

## 💻 How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Explore the dashboard layout and components

## 📚 What I Learned

- Building complex layouts with **CSS Grid**
- Combining Grid and Flexbox effectively
- Structuring large-scale UI components
- Designing dashboards with clear visual hierarchy
