# ✊✋✌ Rock Paper Scissors

A classic Rock Paper Scissors game built with JavaScript. This project focuses on core programming logic, user interaction, and game flow.

## 🚀 Features

* Play Rock, Paper, Scissors against the computer
* Random computer choice generation
* Game logic to determine the winner
* Score tracking system
* Interactive gameplay via the browser console

## 🛠️ Technologies Used

* HTML5 for basic structure
* JavaScript (ES6) for game logic

## 💻 How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Play the game

## 📚 What I Learned

* Writing functions and organizing code
* Using conditionals to control logic flow
* Generating random values in JavaScript
* Handling user input and game states
