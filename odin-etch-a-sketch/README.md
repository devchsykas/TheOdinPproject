# 🎨 Etch-a-Sketch

A browser-based drawing app inspired by the classic Etch-a-Sketch. This project focuses on DOM manipulation, dynamic grid creation, and interactive user input.

## 🚀 Features

* Dynamic grid generation
* Hover-based drawing effect
* Adjustable grid size
* Clear/reset grid functionality
* Interactive and responsive behavior

## 🛠️ Technologies Used

* HTML5 for structure
* CSS3 for styling
* JavaScript (ES6) for interactivity and DOM manipulation

## 💻 How to Use

1. Clone or download this repository
2. Open `index.html` in your browser
3. Move your mouse over the grid to start drawing

## 📚 What I Learned

* Handling events like mouse hover
* Managing application state (grid size, reset, etc.)
* Improving problem-solving through interactive features
